
# yapf (via pipx) (yapf)

yapf is a formatter for Python files.

## Example DevContainer Usage

```json
"features": {
    "ghcr.io/devcontainers-contrib/features/yapf:2": {}
}
```

## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| version | Select the version to install. | string | latest |
